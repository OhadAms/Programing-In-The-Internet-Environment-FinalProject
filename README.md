## ğŸ“ Project Summary

This project provides a **flexible engine for running graph algorithms on matrices**.  
A matrix is treated as a graph where each cell is a node connected to its valid neighbors.

### What it includes:
- **Graph Algorithms:** DFS, BFS, Dijkstra, Connected Components, Island Counting
- **Multithreading:** Algorithms can run in parallel for improved performance
- **Clientâ€“Server Architecture:** A server processes algorithm requests from clients
- **Core Structures:** Matrix, Index, Node, TraversableMatrix (defines source/destination + graph rules)

### What it can do:
- Find reachable nodes
- Detect and count isolated regions
- Compute shortest paths
- Run queries via a client-server setup
